# Changelog

## 1.0.0 - 2026-05-28

Initial release candidate.

- Added one-click on/off popup.
- Added floating on-page feedback panel.
- Added element click selection and grouped selection.
- Added drag-to-select for multiple elements.
- Added Markdown report export.
- Added copy, copy-and-clear, clear, color, auto-copy, and text-capture settings in the page panel.
- Added local-only Chrome storage.
- Added release icon and Chrome Web Store publishing materials.
